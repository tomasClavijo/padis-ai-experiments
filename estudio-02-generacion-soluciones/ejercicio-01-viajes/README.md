# Consigna — Organización de viajes

## Objetivos

- Uso de elementos HTML para el ingreso de información.
- Implementar validaciones de datos en la interfaz.
- Utilizar componentes y estilos Bootstrap.

## Parte A: Ajustes en Formulario HTML

- Incluir un campo que permita ingresar el nombre del destino con la etiqueta 'Destino'.
- Implementar validación de datos con HTML:
  - El campo 'Destino' es requerido.
  - El campo 'Fecha de viaje' deberá ser de tipo fecha (`date`).
  - El campo 'Costo' no debe aceptar valores negativos.
- Implementar dos botones de opción (radio buttons) que incluyan los siguientes medios de transporte:
  - Avión
  - Ómnibus

## Parte B: Bootstrap

- Estilizar el botón de envío con la clase correspondiente al botón primario de Bootstrap.
- Aplicar la clase `form-control` a los inputs del formulario (exceptuando los botones de opción) y la clase `form-label` a las etiquetas correspondientes a cada input.
- Agregar la clase `mx-3` al elemento card para aumentar el margen lateral.
